# Static Site Builder

## Description
This is the static site generator I build to aid in creating static sites.

## Requirements:
- Yarn (^ v0.24.6)
- Gulp (^ v3.9.1)
- Gulp-CLI (^ v1.2.2)

## Build Instructions:
1. `yarn install`
2. `gulp build --dev` or `gulp build --production`
3. $$Profit$$

### Other Commands:
`gulp clean`: Clean dist folder 

`gulp watch`: Watch files and build
